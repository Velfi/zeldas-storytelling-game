# Runtime sound cues

These 44.1 kHz stereo Ogg Vorbis files are gameplay-ready derivatives of the licensed sources under `assets/sound-effects/torn-appointment-priority/`. They are decoded to mono by the runtime. Source provenance and full license records remain beside each source asset.

| Runtime file | Cue | Source |
|---|---|---|
| `clue-revealed.ogg` | Evidence or clue revealed | Freesound `Mystery_chime.ogg` by Unaxete, public preview, CC0; trimmed to 2.2 seconds |
| `fact-established.ogg` | Fact/check established | OpenGameArt `bell_01.ogg` by rubberduck, CC0 |
| `decisive-clue.ogg` | Decisive physical proof | Mixkit `Arabian mystery harp notification`, SFX 2489; trimmed to 3 seconds |
| `reveal-section-proven.ogg` | Final reveal section proven | OpenGameArt `gong_02.ogg` by rubberduck, CC0 |
| `wood-open.ogg` | Door or wooden mechanism opening | Mixkit `Creaky door open`, SFX 195 |
| `wood-close.ogg` | Door or shutter closing | Mixkit `Creaky closing wood door`, SFX 189 |
| `crank-resistance.ogg` | Shutter crank resistance | OpenGameArt `grind_metal.wav` by laleksic, CC0 |
| `candle-extinguished.ogg` | Investigation action/candle spent | Freesound `candle-extinguish.wav` by Krokulator, public preview, CC0 |

`christmas-reveal-tones.wav` is intentionally not used by runtime code.
